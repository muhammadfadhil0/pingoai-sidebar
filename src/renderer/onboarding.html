Quick Guide

How to use PingoAI easily:

  1. Copy the text you want to process with Ctrl+C
  2. Press Ctrl+W to display the PingoAI popup if it doesn't appear automatically
  3. Select the tool you want to use from the popup
  4. Use Ctrl+Q to open or close the chat window
💡 Tips: You can change keyboard shortcuts in the Settings menu anytime.
